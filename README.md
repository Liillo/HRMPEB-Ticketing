# 🎟 HRMPEB Ticketing System

A modern **Laravel-based event ticketing and payment system** built for HRMPEB.  
It supports **M-Pesa STK Push**, **QR code tickets**, **PDF ticket downloads**, and a full **admin validation dashboard**.

---

## 🚀 Features

### 🎫 Ticketing
- Unique UUID ticket numbers
- QR code for each ticket
- Downloadable PDF tickets
- Email ticket delivery after payment

### 💳 Payments
- M-Pesa STK Push integration
- Checkout Request ID storage
- Real-time payment confirmation
- Waiting screen until payment is verified

### 👥 Group Bookings
- Corporate/Group ticket
- Fixed price: **KES 40,000**
- Covers up to **8 people**

### 🛠 Admin Panel
- Create & manage events
- View bookings and payments
- Validate tickets using QR code or ticket number
- View ticket details

### 📧 Notifications
- Automatic email with PDF ticket and QR code
- Payment status alerts

---

## 🧱 Tech Stack

- **Laravel 12**
- PHP 8+
- MySQL
- M-Pesa Daraja API
- Barryvdh DomPDF
- Simple QR Code
- Guzzle HTTP
